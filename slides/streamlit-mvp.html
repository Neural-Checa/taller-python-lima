Streamlitde un script de Python a una app web, en minutos

La forma más rápida de poner una idea —o un LLM— frente a un usuario. Sin HTML, sin JavaScript: solo Python.

Tema: Prototipos y MVPs con IADuración: ~10 min + práctica

El qué

¿Qué es Streamlit?

Una librería de Python que convierte tu script en una aplicación web, sin que escribas una línea de frontend.

  • Cero frontend — ni HTML, ni CSS, ni JavaScript.
  • Cada elemento es una línea — st.button(), st.slider(), st.chat_input().
  • Corres y listo — streamlit run app.py y se abre en tu navegador.

El único concepto que debes entender

El modelo mental

Cada vez que el usuario interactúa (un clic, una tecla), Streamlit vuelve a ejecutar tu script de arriba a abajo.

Interacción
→
Se re-ejecuta
TODO el script
→
Se redibuja
la pantalla

¿Y cómo recuerdas algo entre re-ejecuciones (un contador, un historial de chat)? Con st.session_state. Es el error #1 de quien empieza — y el "aha" #1.

Ordenar la pantalla

Layout en segundos

  • Columnas — st.columns(3) para poner cosas lado a lado.
  • Pestañas — st.tabs([...]) para separar secciones.
  • Barra lateral — st.sidebar para filtros y configuración.
  • Expander — st.expander() para esconder lo secundario (como "Ver el SQL").

De datos a visual

Tablas y gráficos, gratis

Streamlit dibuja directo desde un DataFrame de pandas. Sin configurar nada.

import pandas as pd
import streamlit as st

df = pd.read_csv("ventas.csv")
st.dataframe(df)        # la tabla
st.bar_chart(df)        # el gráfico

→ Mira 3_datos_y_charts.py (incluye subir tu propio CSV).

El momento bueno

+ un LLM = un MVP de IA

import streamlit as st
from langchain_groq import ChatGroq

llm = ChatGroq(model="llama-3.3-70b-versatile")

prompt = st.text_area("Pregunta:")
if st.button("Enviar"):
    st.write_stream(t.content for t in llm.stream(prompt))

Eso de arriba ya es una app de IA completa. → app 4_primer_llm.py.

Criterio

¿Cuándo usar Streamlit?

Sí, ideal para

  • Prototipos y MVPs para validar una idea.
  • Demos y pruebas de concepto con LLMs.
  • Dashboards y herramientas internas.
  • Mostrarle algo a un cliente, ya.

Mejor otra cosa si

  • Es una app de producción de alto tráfico.
  • Necesitas una UI/UX móvil muy pulida.
  • Quieres control fino de cada pixel.

Compártelo con el mundo

Deploy gratis

  • Streamlit Community Cloud — conectas tu repo de GitHub y te da un link público.
  • Gratis — sin servidores ni configuración de infraestructura.
  • Tu API key va en Secrets — nunca en el código (recuerda: el repo es público).

⚠️ Ojo: si despliegas una app con TU key, quienes la usen consumen TU cuota. Para experimentar, mejor que cada uno corra local con su propia key.
